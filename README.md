# Uber Ride Analytics Dashboard (Power BI)

## Project Overview
This project presents an interactive Power BI dashboard analyzing Uber ride data to monitor bookings, revenue trends, vehicle performance, and customer experience metrics.

## Dashboard Features
- Booking and revenue trend analysis
- Monthly & quarterly dynamic switching
- Vehicle performance insights
- Location-based booking patterns
- Customer & driver ratings analysis
- Interactive KPI monitoring

## Tools & Skills Used
- Power BI
- DAX Measures
- Data Modeling
- Field Parameters
- Interactive Dashboard Design
- Data Cleaning & Transformation

## Key Insights
- Identified peak booking periods
- Analyzed revenue contribution by vehicle types
- Evaluated customer experience trends
